Azure Fundamentals Flashcards

Define Cloud Computing

The on-demand availability of computer system resources, especially data storage and computing power, without direct active management by the user.

Name 3 cloud deployment services

Public - uses cloud provider's data centers

Private - uses their own data centers

Hybrid - 50/50 of the above

How is High Availability achieved?

Data and code stored in the cloud are automatically copied to multiple data centers around the world.

What does Scalability mean?

The application running in the cloud can expand its size when there are more users in the system.

What are the 3 cloud service models?

SaaS - Software as a service

IaaS - Infrastructure as a service

PasS - Platform as a service

What are the 5 Core Cloud services in Azure?
